* README
This project uses Vite + React to build interactive 3D personality
assessment games.

Live: https://ghollisjr.github.io/aboutyou/

Audio files came from ~/Music/MyRecordings/tictachyon_personality/soundfx/export/

** Quick Start

#+begin_src bash
npm install
npm run dev
#+end_src

Open the localhost URL shown by Vite. The retro arcade menu lets you
select a game.

** Build & Preview

#+begin_src bash
npm run build     # produces dist/
npm run preview   # serve built output locally
#+end_src

** Deployment
Pushing to =main= triggers the GitHub Actions workflow
(=.github/workflows/deploy.yml=), which builds and deploys to GitHub
Pages automatically.

** Adding a New Game
1. Create =new-game.jsx= component (with =export default=)
2. Create =src/new-game-main.jsx= entry point (imports component, renders into =#root=)
3. Create =games/new-game.html= shell (=<div id="root">= + =<script type="module">=)
4. Add to =vite.config.js= rollup inputs
5. Update =index.html= card from "Coming Soon" to playable

** Museum Trip Starting Points
The =tripStarts= array in =museum-3d.jsx= defines the random starting
positions/orientations for the psychedelic trip mode. To add a new one:

1. Enter the trip and use the FPS overlay (press =F=) to find a good
   =pos= and =fwd= vector
2. Convert to yaw/pitch:
   - =yaw = atan2(fwd.x, -fwd.z)=
   - =pitch = -asin(fwd.y)=
3. Append to the =tripStarts= array:
   #+begin_src js
   { pos: [x, 2, 0], yaw: ..., pitch: ... },
   #+end_src
   Note: =y= should stay =2= and =z= should stay =0= (X-axis only movement).
