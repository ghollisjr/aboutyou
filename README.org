* README
This project uses Vite + React to build interactive 3D personality
assessment games.

Live: https://ghollisjr.github.io/aboutyou/

** Quick Start

#+begin_src bash
npm install
npm run dev
#+end_src

Open the localhost URL shown by Vite. The retro arcade menu lets you
select a game.

** Build & Preview

#+begin_src bash
npm run build     # produces dist/
npm run preview   # serve built output locally
#+end_src

** Deployment
Pushing to =main= triggers the GitHub Actions workflow
(=.github/workflows/deploy.yml=), which builds and deploys to GitHub
Pages automatically.

** Adding a New Game
1. Create =new-game.jsx= component (with =export default=)
2. Create =src/new-game-main.jsx= entry point (imports component, renders into =#root=)
3. Create =games/new-game.html= shell (=<div id="root">= + =<script type="module">=)
4. Add to =vite.config.js= rollup inputs
5. Update =index.html= card from "Coming Soon" to playable
